# Legacy Material

This folder is reserved for file-based artifacts from the older OpenClaw-workspace-driven prototype.

Examples:
- JSONL lead trackers
- markdown pipeline boards
- search logs
- old tailored resume files
- historical artifacts and notes

## Rule

Material placed here is:
- useful for reference
- useful for fixtures/tests
- useful for migration planning

It is **not** the canonical runtime state for the hosted Job Ops Console.

Canonical runtime state belongs in:
- PostgreSQL
- object storage
- queue/worker runtime
